डिजिटल सुरक्षा

दुर्व्यवहारी निगरानी, नियंत्रण, और ट्रैकिंग के लिए तकनीक का उपयोग करते हैं। यह मार्गदर्शिका आपको सुरक्षित रहने में मदद करती है।

⚠️ शुरू करने से पहले: यदि आपको लगता है कि आपका फोन मॉनिटर किया जा रहा है, तो बदलाव एक सुरक्षित डिवाइस से करें — library computer, दोस्त का फोन। मॉनिटर किए जाने वाले डिवाइस पर बदलाव करने से दुर्व्यवहारी को अलर्ट हो सकता है।
🕵️ क्या मेरा फोन मॉनिटर किया जा रहा है?+

Spyware के चेतावनी संकेत

  • बैटरी सामान्य से जल्दी खत्म होती है, भले ही फोन idle हो।
  • फोन उपयोग न होने पर भी गर्म हो जाता है।
  • असामान्य डेटा उपयोग — Settings → Mobile Data जाएँ और देखें कौन से apps अत्यधिक डेटा उपयोग कर रहे हैं।
  • फोन को बंद होने में सामान्य से अधिक समय लगता है।
  • दुर्व्यवहारी आपकी बातचीत या गतिविधियों के बारे में ऐसी बातें जानता है जो आपने नहीं बताई।
  • अज्ञात apps — विशेषकर "System Update," "Phone Manager," या parental control apps।

यदि आपको spyware का संदेह हो तो क्या करें

  • अभी apps न हटाएँ — इससे दुर्व्यवहारी को पता चल सकता है। पहले निकास की योजना बनाएँ।
  • एक सुरक्षित डिवाइस से मदद लें।
  • Factory reset से सभी spyware हट जाते हैं — लेकिन यह सुरक्षित जगह पहुँचने के बाद करें।
✅ सबसे सुरक्षित विकल्प: एक नया सस्ता फोन नई SIM के साथ खरीदें। इसे केवल आश्रयों, वकीलों, और विश्वसनीय संपर्कों से communicate करने के लिए उपयोग करें।
🔐 अपने Accounts सुरक्षित करें+

Account सुरक्षा की प्राथमिकता

  1. Email पहले — बाकी सब email से reset हो सकता है। एक सुरक्षित डिवाइस से पासवर्ड बदलें।
  2. WhatsApp — Settings → Account → Two-step verification → एक 6-अंकीय PIN सेट करें।
  3. Banking apps — अपना mobile banking PIN बदलें। यदि संभव हो नई SIM पर registered number बदलें।
  4. Instagram, Facebook — पासवर्ड बदलें, 2-factor authentication चालू करें।

Logged-in sessions जाँचें

  • Google: myaccount.google.com → Security → "Your devices" → अज्ञात device हटाएँ।
  • WhatsApp: Settings → Linked Devices → सभी linked devices हटाएँ।
  • Instagram: Profile → Settings → Security → Login Activity → अज्ञात sessions से log out करें।
💬 सुरक्षित तरीके से संवाद करें+

सुरक्षित messaging apps

  • Signal — सबसे सुरक्षित। Messages end-to-end encrypted हैं और default रूप से cloud backup नहीं होते। "Disappearing messages" चालू करें।
  • WhatsApp — End-to-end encrypted, लेकिन Google Drive backup को दुर्व्यवहारी पढ़ सकता है यदि उसके पास आपके Google account की पहुँच है।

सुरक्षित browsing

  • किसी भी browser पर private/incognito mode उपयोग करें — device पर history save नहीं होती।
  • घर का Wi-Fi router websites की list रख सकता है — sensitive topics research करते समय mobile data (घर का Wi-Fi नहीं) उपयोग करें।
  • किसी shared device पर इस site को visit करने के बाद history हटाएँ: Chrome → ⋮ → History → Clear browsing data.

Location सुरक्षा

  • WhatsApp पर Live Location sharing बंद करें: Contact → View Live Location → Stop sharing.
  • दुर्व्यवहारी को Google Maps location sharing से हटाएँ।
  • Photos में GPS metadata होता है — Camera app के लिए location बंद करें।
🇮🇳 भारत-विशिष्ट डिजिटल खतरे+

आधार का दुरुपयोग

  • अपनी आधार biometrics lock करें: uidai.gov.in → myAadhaar → Lock/Unlock Biometrics।
  • अपनी आधार authentication history जाँचें: uidai.gov.in → Aadhaar Authentication History।

SIM swap धोखाधड़ी

  • दुर्व्यवहारी आपके ID का उपयोग करके carrier से duplicate SIM माँग सकता है और आपके OTPs प्राप्त कर सकता है।
  • अपने phone पर SIM lock PIN सेट करें।

Truecaller जोखिम

  • यदि दुर्व्यवहारी आपका नाम जानता है, तो वह Truecaller पर आपका नया नंबर खोज सकता है।
  • Truecaller opt-out करें: truecaller.com/unlisting
✅ यदि डिजिटल सबूत आपके खिलाफ उपयोग किए जा रहे हैं (screenshots, fake profiles, morphed images), तो cybercrime.gov.in पर शिकायत करें या 1930 पर कॉल करें।

🔒 गोपनीयता: कोई कुकीज़ नहीं, कोई ट्रैकर नहीं।

अस्वीकरण: यह सामान्य जानकारी है — कानूनी सलाह नहीं। तत्काल खतरे में 112 या 181 पर तुरंत कॉल करें।
